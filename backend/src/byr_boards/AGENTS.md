# byr_boards

## 作用范围

本目录只负责论坛板块列表页的请求与解析。

负责：

1. 板块页请求。
2. 板块页元信息解析。
3. 帖子列表解析。

不负责：

1. 登录。
2. cookie 管理。
3. 帖子详情页解析。

## 需要知道的外部知识

这些知识只和板块列表页采集有关，不应写回项目顶层。

1. 用户在浏览器里访问的地址可能是 `https://bbs.byr.cn/#!board/IWhisper` 这种 hash 路由形式。
2. 实际可直接请求的板块内容接口是 `/board/{board_name}`。
3. 当前已验证板块页请求可携带参数 `p=<页码>` 和 `_uid=<当前登录用户 id>`。
4. 当前已验证悄悄话板块第一页可以通过 `/board/IWhisper?p=1&_uid=<uid>` 获取到真实列表内容。
5. 板块页返回内容中包含 `.board-list` 表格，可用于提取帖子列表。
6. 板块页中可以解析出在线人数、今日帖数、分页信息、帖子列表等内容。
7. 置顶帖和普通帖会同时出现在板块列表中；是否保留、过滤、单独标记，属于板块模块内部的输出策略问题。

## 工作规范

当前目录下暂不新增额外规范，后续遇到真实问题再补充。

## 禁止项

1. 不要在这里保存登录逻辑。
2. 不要在这里保存帖子详情页楼层解析逻辑。
