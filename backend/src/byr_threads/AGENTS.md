# byr_threads

## 作用范围

本目录只负责论坛帖子详情页的请求与解析。

负责：

1. 帖子详情页请求。
2. 帖子分页信息解析。
3. 主贴与回复楼层解析。

不负责：

1. 登录。
2. cookie 管理。
3. 板块列表页抓取。

## 需要知道的外部知识

这些知识只和帖子详情页采集有关，不应写回项目顶层。

1. 实际可直接请求的帖子详情页接口是 `/article/{board_name}/{article_id}`。
2. 当前已验证帖子详情页请求可携带参数 `p=<页码>` 和 `_uid=<当前登录用户 id>`。
3. 帖子详情页中每一层内容对应一个 `.article` 区块。
4. 楼层作者显示名位于 `.a-u-name`，悄悄话里既可能是 `IWhisper#123` 这样的匿名编号，也可能是真实昵称。
5. 帖子正文位于 `.a-content-wrap`，其中包含发信人、标题、发信站等头部元信息，需要和真正正文区分。
6. 帖子页中会同时出现分页区块和楼层操作按钮；解析正文时不要把这些交互元素当作帖子内容。

## 工作规范

当前目录下暂不新增额外规范，后续遇到真实问题再补充。

## 禁止项

1. 不要在这里保存登录逻辑。
2. 不要把板块列表页解析逻辑写进帖子目录。
