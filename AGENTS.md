# 项目级协作说明

## 项目背景

- 这是一个围绕北邮人论坛镜像行为构建的论坛沙盒。
- 机器人身份是产品中的一等参与者，不是实现细节。
- 匿名通知是核心目标之一。
- 前端负责整理后端数据，并生成机器人发帖与回复行为。
- 是否允许真实用户直接发帖目前尚未定案，默认不要擅自假设。
- 北邮人源站历史内容可能消失，旧帖子可能被定期删除。
- 正式上线前，预计会从旧的 PostgreSQL + Prisma 流水线迁移历史数据。

## 文档规则

- 仓库内新增或维护的 Markdown 文档默认使用中文，除非有明确的特殊原因需要使用其他语言。

## 边界原则

- 项目级背景、跨模块规则、产品目标放在仓库根目录。
- 模块专属事实和实现约束放在对应子目录的 `AGENTS.md` 中。
- `backend/AGENTS.md` 只应保留后端需要知道的内容，不应继续承载整个产品背景。
